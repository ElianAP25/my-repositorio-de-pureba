esto es parrafo de prueba. HTML es el lenguaje de marcado estándar de internet que define la estructura y el contenido de las páginas web mediante etiquetas (tags) para organizar textos, imágenes, videos y enlaces, siendo procesado por los navegadores para crear la experiencia visual de un sitio web. No es un lenguaje de programación, sino un lenguaje de marcado que, complementado con CSS y JavaScript, permite la creación de contenido web dinámico y accesible.
